# Reversi

This is reversi game wriiten in rust.

### Install

```
git clone https://github.com/masan4444/reversi.git
cd reversi
cargo build --release
```

### Run

```
cargo run --release
```

### Test

```
cargo test
```

### Bench

```
cargo bench
```

## Install rust and cargo

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup install nightly
```
